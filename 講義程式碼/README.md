# 講義程式碼：每一段的完整可執行檔

這個資料夾對應講義《動手做：親手打造你的微型 AI 智慧體》，
每一關的完整程式都在這裡，內容跟講義裡的程式**一字不差**、全部實際執行測試過。

## 使用前

1. 把你填好金鑰的 `.env` 複製到**這個資料夾**裡（跟 .py 檔同一層）。
2. 之後直接執行，例如：`py step3_第一則訊息.py`

## 檔案對照

| 檔案 | 對應講義 | 內容 |
|---|---|---|
| `step3_第一則訊息.py` | STEP 3 | 最小 chatbot：問一句、答一句 |
| `step4_多輪對話.py` | STEP 4 | 有記憶的連續聊天（/exit 離開） |
| `step5_ReAct搜尋.py` | STEP 5 | 手動轉一圈 ReAct（搜尋工具） |
| `step6_整合agent.py` | STEP 6 | 自動迴圈，完整可用的智慧體 ✅ |
| `step7_終端機工具.py` | STEP 7 | 加上終端機工具（教學版） |
| `agent_最終定版.py` | STEP 7.5 | 修好 3 個 bug 的實測穩定版 ⭐ 建議用這份 |
| `進階_分層版/` | 附錄 B | 拆成 main.py / ai.py / tools.py 的專業架構 |

## 進階_分層版怎麼跑

```powershell
cd 進階_分層版
# 也要把 .env 複製進去這一層
py main.py
```

## ⚠️ 注意

- `.env` 裡的金鑰是密碼。**要把這個資料夾分享給別人之前，先把 `.env` 刪掉。**
- 如果看到「呼叫模型失敗…429」，是免費額度暫時被打滿，等幾秒再問一次就好。

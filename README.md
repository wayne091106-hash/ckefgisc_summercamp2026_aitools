# 暑訓講義：AI工具

不是教你用 ChatGPT，是自己動手寫一個會查資料、會操作電腦的 AI agent。

## 這個 repo 有什麼

- [`暑訓講義_AI工具.md`](暑訓講義_AI工具.md) —— 完整講義，STEP 1 到 STEP 7.5，每一段程式碼先拆解再組裝，可以自己讀、卡住有解答
- [`講義程式碼/`](講義程式碼) —— 每一關的完整可執行檔，跟講義裡的程式一字不差、全部實測過

## 其他版本

- 投影片（課堂用）：[Slides.com](https://slides.com/ck11400667/aitools)
- 線上詳細講義：[HackMD](https://hackmd.io/@FObDA6rXRKSTsKAP5phQWw/HyEjQD5XGg)

## 快速開始

1. 進 [`講義程式碼/`](講義程式碼) 資料夾
2. 複製 `.env.example` 改檔名為 `.env`，依講義 STEP 1～2 申請一組 API 金鑰填進去
3. `py step3_第一則訊息.py` 開始，一路跑到 `agent_最終定版.py`

⚠️ 金鑰是密碼，`.env` 不要傳給別人、不要上傳。
